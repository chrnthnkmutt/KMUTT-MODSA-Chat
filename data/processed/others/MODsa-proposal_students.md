โครงการพัฒนาแชทบอทอัจฉริยะ MOD-SA1 ด้วยเทคนิค Large Language Models ร่วมกับ
Retrieval-Augmented Generation (LLMs–RAG)
จัดทำโดย ผศ.ดร.ณัฐนาถ เหมือนสุวรรณ ภาควิชาวิศวกรรมคอมพิวเตอร์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี
ที่ปรึกษา ศ.ดร.บุญเจริญ ศิริเนาวกุล
1. หลักการและเหตุผล
ในปัจจุบันนักศึกษามหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี (KMUTT) ตลอดจนนักเรียนหรือผู้ที่สนใจเข้าศึกษาต่อ และ
ผู้ปกครองต้องค้นหาเพื่อเข้าถึงข้อมูลด้านการศึกษา การลงทะเบียน ระเบียบข้อบังคับ ทุนการศึกษา และบริการต่าง ๆ ของ
มหาวิทยาลัยจากหลายแหล่งข้อมูล ซึ่งอาจทำให้เกิดความสับสนและใช้เวลานานในการค้นหา และมีการติดต่อสอบถามข้อมูลกับ
เจ้าหน้าที่ของทางมหาวิทยาลัยที่ให้บริการเข้ามาเป็นจำนวนมาก ระบบแชทบอทจะสามารถแก้ปัญหานี้ได้โดยตอบคำถามให้กับผู้ใช้
ได้อัตโนมัติตลอด 24 ชั่วโมง ลดภาระงานของเจ้าหน้าที่ทำให้สามารถใช้เวลาไปในการพัฒนางานหรือปฎิบัติงานส่วนอื่นได้มากขึ้น
ระบบแชทบอทนี้จะพัฒนาโดยใช้เทคโนโลยีหลักคือ Large Language Models (LLMs) ร่วมกับเทคนิค Retrieval-Augmented
Generation (RAG) เป็นแนวทางที่ช่วยให้ระบบแชทบอทสามารถตอบคำถามได้อย่างถูกต้อง อ้างอิงข้อมูลจริงขององค์กร และลด
ปัญหาการสร้างข้อมูลผิดพลาด (hallucination) โดยดึงข้อมูลจากแหล่งความรู้ของทางหน่วยงานที่จัดเตรียมไว้
โครงการนี้มีเป้าหมายเพื่อพัฒนาแชทบอทอัจฉริยะสำหรับนักศึกษามจธ.
ตลอดจนนักเรียนหรือผู้ที่สนใจเข้าศึกษาต่อ
และ
ผู้ปกครอง โดยมีอาจารย์และนักศึกษาจากภาควิชาวิศวกรรมคอมพิวเตอร์ คณะวิศวกรรมศาสตร์ มจธ. ร่วมมือกับฝ่ายพัฒนา
นักศึกษาและบริการ ในการจัดเตรียมข้อมูล พัฒนา และทดสอบระบบ เพื่อตอบสนองความต้องการของมหาวิทยาลัยไปพร้อมกับ
พัฒนาทักษะและประสบการณ์ในการลงมือวิจัยและพัฒนาของนักศึกษาไปพร้อมกัน
2. วัตถุประสงค์ของโครงการ
1. เพื่อพัฒนาแชทบอทสำหรับนักศึกษามจธ. ผู้ปกครอง และ ผุ้ที่ประสงค์จะเข้าศึกษาต่อ โดยใช้เทคนิค LLMs–RAG
2. เพื่อรวบรวมและจัดเตรียมฐานความรู้ (Knowledge Base) ที่เกี่ยวข้องกับบริการนักศึกษา
3. เพื่อให้นักศึกษาได้รับประสบการณ์จริงด้านพัฒนาระบบ AI, NLP และระบบแชทบอท
4. เพื่อเพิ่มช่องทางการเข้าถึงข้อมูลของนักศึกษาอย่างสะดวกและรวดเร็ว ลดภาระงานของเจ้าหน้าที่ที่เกี่ยวข้อง
3. กลุ่มเป้าหมาย
• นักศึกษาปัจจุบันของมหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี
• ผู้ปกครองของนักศึกษาปัจจุบัน
• ผู้ที่ประสงค์จะเข้าศึกษาต่อในระดับต่างๆ พร้อมผู้ปกครอง
• เจ้าหน้าที่ บุคลากรที่เกี่ยวข้องกับงานบริการนักศึกษา (ใช้เป็นเครื่องมือสนับสนุน)

5. แนวทางและเทคโนโลยีที่ใช้ (Methodology)
• Large Language Models (LLMs):  เป็นเทคโนโลยีปัญญาประดิษฐ์ที่สามารถเข้าใจและประมวลผลภาษามนุษย์ เมื่อ
นำมาใช้ในการพัฒนาแชทบอท
จะช่วยให้ระบบสามารถเข้าใจคำถามของผู้ใช้และตอบคำถามได้อย่างถูกต้อง
เป็น
ธรรมชาติ และสอดคล้องกับบริบท โดยทำหน้าที่เป็นกลไกหลักในการสื่อสารระหว่างผู้ใช้กับระบบแชทบอท
• Retrieval-Augmented Generation (RAG): เป็นเทคนิคที่ช่วยให้แชทบอทสามารถค้นหาข้อมูลที่เกี่ยวข้องจาก
ฐานข้อมูลหรือเอกสารที่จัดเตรียมไว้ ก่อนนำข้อมูลดังกล่าวมาใช้ร่วมกับปัญญาประดิษฐ์ในการสร้างคำตอบ วิธีการนี้ช่วย
ให้แชทบอทตอบคำถามได้อย่างถูกต้อง อ้างอิงข้อมูลจริง และลดความคลาดเคลื่อนของข้อมูลได้อย่างมีประสิทธิภาพ
• เครื่องมือที่ใช้ (สามารถปรับเปลี่ยนได้เพื่อความเหมาะสม):
o Python
o LangChain / LlamaIndex
o Vector Database
o Tool for web-based Chat UI
6. ประโยชน์ที่จะได้รับ
การพัฒนาแชทบอทอัจฉริยะสำหรับนักศึกษามหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี จะก่อให้เกิดประโยชน์ต่อทั้งผู้ใช้งานและ
หน่วยงานผู้ให้บริการ ดังนี้
6.1 แชทบอทสามารถให้บริการตอบคำถามแก่ผู้ใช้งานได้ตลอด 24 ชั่วโมง ไม่จำกัดวันและเวลาราชการ ทำให้นักศึกษาและ
ผู้ปกครองสามารถเข้าถึงข้อมูลด้านการเรียน การลงทะเบียน ระเบียบข้อบังคับ และบริการต่าง ๆ ของมหาวิทยาลัยได้อย่างสะดวก
รวดเร็ว และทันท่วงที เพิ่มประสิทธิภาพในการเข้าถึงข้อมูลและลดข้อจำกัดด้านเวลาในการติดต่อหน่วยงาน
6.2 แชทบอทสามารถช่วยลดภาระงานของเจ้าหน้าที่ผู้ให้บริการ โดยเฉพาะการตอบคำถามที่มีลักษณะซ้ำซ้อนหรือเป็นคำถามที่พบ

8. งบประมาณดำเนินโครงการ (Budget Plan)
โครงการพัฒนาแชทบอทอัจฉริยะสำหรับนักศึกษามจธ. ด้วยเทคนิค LLMs–RAG มีงบประมาณหลักแบ่งออกเป็น 3 ส่วน ได้แก่
1. ค่าจ้างนักศึกษา
2. ค่าใช้จ่ายด้าน Model AI (Tokens / API) และ GPU
3. ค่าใช้จ่ายด้านระบบ Cloud และโครงสร้างพื้นฐาน
9. ตัวชี้วัดความสำเร็จ

• การส่งมอบระบบและเอกสารครบถ้วนตามแผน
